## Unit Tests

### Write Unit Tests

Write tests alongside components and services, as `<thing>.spec.ts`:
```
components/
 - About.ts
 - About.spec.ts
```

### Run Tests with Karma
```
$ npm test
```

Or

```
$ npm install -g karma
$ karma start
```

### Run the Universal server locally:

```
$ gulp
$ npm run start_universal
```

Then navigate to localhost:3000
